#  AI Text-to-Image Generator

This project generates images from text prompts using Stable Diffusion.

##  Built Using
- Python
- Stable Diffusion
- Gradio
- Google Colab

##  Live Demo
https://aca67ba9d1217ea156.gradio.live

##  Features
- Text → Image generation
- Style selection
- Multiple images
- Negative prompts

##  Project File
- Colab Notebook: `Text_to_image.ipynb`
